Introduction to: how the internet works, how to build web sites, and the multiple options available for building websites. Includes HTML, CSS, Javascript, web servers, and server-side programming. Prerequisite: CSCI 1614.


  • Focus on vanilla HTML, CSS, JavaScript.
  • Learn the ins and outs of technologies that drive the modern web.
  • Following The Odin Project curriculum
  • Hands-on learning with practical projects
  • Building strong fundamentals for web development (Necessary for vibe-code-god-dom)
