üìã Quick Reference

Common Patterns:

  • Check if exists: if (variable) {...}
  • Default value: let x = input || "default"
  • Range check: if (x >= min && x <= max)

Debugging Tips:

  • Use console.log() liberally
  • Check typeof variable
  • Test truthy/falsy with !!value
