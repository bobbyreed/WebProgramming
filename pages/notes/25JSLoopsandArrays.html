The Big Three Array Methods

Method Purpose Returns

Map Method

Map _______________ each element and returns a _______________ array.

const numbers = [1, 2, 3, 4, 5]; const doubled = numbers.map(num => _______________); // Result: [2, 4, 6, 8, 10]

Filter Method

Filter returns elements where the callback returns _______________.

const numbers = [1, 2, 3, 4, 5, 6]; const evens = numbers.filter(num => _______________); // Result: [2, 4, 6]

Reduce Method

Reduce combines all elements into a _______________ value.
