For Loops ğŸ¢

  console.log(`Iteration ${i}`);
}

Parts of a For Loop:

  1. Initialization: let i = 0
  2. Condition: i < 5
  3. Update: i++

Execution Order:

  1. Initialize once
  2. Check condition
  3. Execute body
  4. Update
  5. Go to step 2

The Filter Method ğŸ”

Filter returns elements where callback returns true

The Reduce Method ğŸ“Š
