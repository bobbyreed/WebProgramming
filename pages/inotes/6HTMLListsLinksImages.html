PART 1: LISTS (Slides 3-7) - 10 minutes

Slide 3: Lists Introduction (2 min)
Opening Hook:
  • Ask: "Who's made a to-do list today?" - Connect to real life
  • Show website examples: Amazon (shopping lists), Netflix (watch lists)
  • Emphasize: Lists are THE most common HTML element after paragraphs
KEY POINT: Two types - Unordered (bullets) and Ordered (numbers)
Slide 4: Unordered Lists (2 min)
LIVE DEMO - Type this with students: <ul> <li>Milk</li> <li>Eggs</li> <li>Bread</li> <li>Coffee</li> </ul>
Teaching Points:
  • ul = "Unordered List"
  • li = "List Item"
  • Show in browser immediately
  • Point out automatic bullets
Slide 5: Ordered Lists (2 min)
Quick Switch Demo: Change the shopping list from <ul> to <ol> to show instant numbering
Emphasize:
  • Use for steps, rankings, sequences
  • Numbers automatically update if you reorder items
  • Ask: "What would happen if we deleted step 2?" (numbers auto-adjust!)
Slide 6: Lists Side by Side (2 min)
Quick Quiz Questions:
  1. Recipe ingredients? → Unordered
  2. Recipe steps? → Ordered
  3. Navigation menu? → Unordered
  4. Top 10 movies? → Ordered
Common Mistake: Students forget <li> tags - emphasize BOTH are needed!
Slide 7: Nested Lists (2 min)
Build Together: Create a grocery list with categories
Key Points:
  • Indent nested lists for readability
  • Can mix ordered inside unordered (and vice versa)
  • No limit to nesting depth (but be reasonable)

PART 2: LINKS (Slides 8-14) - 15 minutes

Slides 8-9: Links Introduction & Anchor Element (3 min)
The "Web" Concept:
  • Draw a web on the board - show interconnected pages
  • Links are the threads that connect everything
  • Without links, we'd have isolated pages
CRITICAL DEMO - Show broken vs working link: <a>This doesn't work</a> <a href="https://www.google.com">This works!</a>
STRESS: href attribute is REQUIRED for links to work!
Slide 10: HTML Attributes (2 min)
Attribute Structure:

attribute="value" - always in opening tag!

Common Attributes We'll Use Today:
  • href - hypertext reference (links)
  • src - source (images)
  • alt - alternative text (accessibility)
  • target - where to open
  • rel - relationship (security)
Slide 11: Absolute vs. Relative Links (3 min)
Quick Reference:
  • Absolute: https://www.site.com/page.html
  • Relative same folder: page.html
  • Relative subfolder: folder/page.html
  • Relative parent: ../page.html
Analogy: Absolute = full street address | Relative = "next door" or "upstairs"
Slide 12: File Path Navigation (3 min)
PROJECT STRUCTURE TO DRAW: project/ ├── index.html (YOU ARE HERE) ├── about.html ├── pages/ │ ├── contact.html │ └── portfolio.html └── images/ └── logo.png
Practice Together:
  1. From index to contact: pages/contact.html
  2. From contact to index: ../index.html
  3. From contact to logo: ../images/logo.png
Common Error: Forgetting ../ to go up directories
Slides 13-14: Link Security (4 min)
SECURITY DEMONSTRATION:

Show tabnabbing attack diagram (draw on board)

  1. User clicks link with target="_blank"
  2. New tab opens malicious site
  3. Malicious site changes original tab
  4. User returns to phishing page
Security Solution:

rel="noopener noreferrer"

  • noopener = blocks window.opener
  • noreferrer = hides referrer + includes noopener
Always use this pattern: target="_blank" rel="noopener noreferrer"

PART 3: IMAGES (Slides 15-19) - 15 minutes

Slides 15-16: Images & Attributes (4 min)
BASIC IMAGE SYNTAX: <img src="path/to/image.jpg" alt="Description"> WITH SIZE: <img src="image.jpg" alt="Cat" width="400" height="300">
CRITICAL Points:
  • img is VOID element - no closing tag!
  • BOTH src and alt are REQUIRED
  • alt text is NOT optional (accessibility law!)
Why alt text matters - Personal story:

Share example of screen reader user or slow connection scenario

Slide 17: Image Formats (3 min)
Format Quick Guide:
Use Case Format Why
Photos JPEG Best compression for photos
Logos PNG or SVG Transparency/scalability
Simple animations GIF Universal support
Icons SVG Tiny file size, scales perfectly
Slide 18: Image Organization (3 min)
CREATE THIS STRUCTURE LIVE:
  1. Create images folder
  2. Add subfolders for organization
  3. Show good naming: "team-photo.jpg" not "IMG_1234.jpg"
File names are case-sensitive on most web servers! photo.jpg ≠ Photo.jpg
Slide 19: Image Copyright (5 min)
LIVE DEMO - Find Free Images:
  1. Go to Unsplash.com
  2. Search for "coffee"
  3. Download and add to project
  4. Show how to give attribution
Free Image Sites:
  • Unsplash.com - No attribution required
  • Pexels.com - No attribution required
  • Pixabay.com - Check individual licenses
  • Google Images → Tools → Usage Rights → Creative Commons
LEGAL WARNING: Using copyrighted images = potential lawsuit!

TROUBLESHOOTING GUIDE

Problem Quick Fix Prevention
Links not working Check href attribute exists Always test links
Images not showing Check path and file extension Use lowercase, no spaces
Lists look wrong Ensure <li> inside <ul>/<ol> Proper nesting
Path confusion Draw folder structure Use ./ for clarity
Copyright concerns Use Unsplash/Pexels Bookmark free sites
