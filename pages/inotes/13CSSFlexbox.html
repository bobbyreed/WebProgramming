UNDERSTANDING AXES (Minutes 25-28)

3 minutes
Slide 9: Main and Cross Axis
This concept is FUNDAMENTAL!
Use hand gestures: horizontal for main (row), vertical for cross
Students fill in:

Memory trick: "justify-MAIN, align-CROSS" - repeat 3 times with class
Key rule: justify-content = MAIN axis, align-items = CROSS axis

COMMON STUDENT QUESTIONS

  1. Q: "Why isn't my item flexing?"
    A: Check if it's a direct child. Grandchildren don't flex!
  2. Q: "What's the difference between align-items and align-content?"
    A: Save align-content for when we cover flex-wrap next class
  3. Q: "Can I use Flexbox and Grid together?"
    A: Yes! Grid for layout, Flexbox for components (we'll cover Grid later)
  4. Q: "Why is my item shrinking when I don't want it to?"
    A: Default flex-shrink is 1. Set flex-shrink: 0 to prevent
