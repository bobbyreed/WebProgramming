LIVE CODING DEMO (Minutes 37-42)

5 minutes
Slide 18: Live Coding Demo
BUILD A MINI BLOG POST:
  1. Start with h1: "My First Blog Post"
  2. Add h2: "Introduction"
  3. Add paragraph with lorem ipsum
  4. Add <strong> to emphasize a word
  5. Add another h2: "Main Content"
  6. Add paragraph with <em> for emphasis
  7. Add h3: "Important Details"
  8. Add comments for each section
  9. Show proper indentation throughout
Go slowly! Students should follow along in their own VS Code

Check for Understanding:

  • Ask: "Why only one h1?"

  • Ask: "Why do we indent?"

Must-Emphasize Concepts:

  1. Only ONE h1 per page - This is critical!
