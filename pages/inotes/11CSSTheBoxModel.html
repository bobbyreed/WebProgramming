The Box Model üì¶

Content

Every box has four parts!

Box Model Components

From Inside to Outside:

  1. Content: The actual content (text, images)
  2. Padding: Space between content and border
  3. Border: The edge of the box
  4. Margin: Space outside the border
Memory Trick: "Content Pads the Border from the Margin" (CPBM)

1. Content üìù

.box {
  width: 300px;
  height: 150px;
}
300px √ó 150px content
  • The innermost part of the box
  • Where your text and images live
  • Size controlled by width and height

2. Padding üõ°Ô∏è

.box {
  padding: 20px;
  /* or individually: */
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 10px;
  padding-left: 20px;
}
  • Creates space INSIDE the border
  • Background color extends into padding
  • Makes content easier to read
  • Can't be negative

Padding Shorthand ‚ö°

4 Values

padding: 10px 20px 30px 40px;
/* top right bottom left */
/* Clockwise from top! */

2 Values

padding: 10px 20px;
/* vertical horizontal */
/* top/bottom left/right */
Remember: Think of a clock! Start at 12 and go clockwise.

3. Border üî≤

.box {
  border: 5px solid #764ba2;
  /* width style color */

  /* or separately: */
  border-width: 5px;
  border-style: solid;
  border-color: #764ba2;
}

Border Styles:

  • solid, dashed, dotted, double, none
  • Adds to the total size of the element
  • Can have different styles per side

4. Margin üåä

.box {
  margin: 20px;
  /* Creates space OUTSIDE the border */

  /* Center horizontally: */
  margin: 0 auto;
}
  • Space between elements
  • Can be negative (pulls elements closer)
  • Transparent - shows parent's background
  • auto value for centering

The Box-Sizing Problem üò§

You set width: 200px, but your box is 260px wide!

.box {
  width: 200px;
  padding: 20px;
  border: 10px solid black;
}

Actual width:

200px (content) + 40px (padding) + 20px (border) = 260px

This is the default content-box behavior!

The Border-Box Solution! ‚ú®

.box {
  box-sizing: border-box;
  width: 200px;
  padding: 20px;
  border: 10px solid black;
}

With border-box:

Total width = 200px (includes padding and border!)

Pro Tip: Width now means the TOTAL width, not just content!

Set Border-Box for Everything! üåç

/* Add this to the top of your CSS! */
*, *::before, *::after {
  box-sizing: border-box;
}

Why use border-box?

  • More intuitive sizing
  • Easier to calculate layouts
  • Industry standard practice
  • Prevents unexpected overflow
Always include this in your projects from now on!

Content-Box vs Border-Box

content-box (default)

width: 200px

Actual: 260px wide

200 + 40 (padding) + 20 (border)

border-box

width: 200px

Actual: 200px wide

Padding and border included!

Margin Collapsing ü§ù

When vertical margins meet, they collapse!

.box1 {
  margin-bottom: 30px;
}

.box2 {
  margin-top: 20px;
}

Space between boxes = 30px (not 50px!)

The larger margin wins!

Note: Only happens with vertical margins, not horizontal!

Centering Elements üéØ

The Classic Technique: margin: auto

.centered-box {
  width: 500px;
  margin: 0 auto;
  /* top/bottom: 0, left/right: auto */
}
Centered!
Requirements: Element must have a defined width!

When to Use What? ü§î

Use Padding

  • Space inside a button
  • Text breathing room
  • Card/container spacing
  • Background needs to extend

Use Margin

  • Space between elements
  • Centering blocks
  • Creating gaps in layouts
  • Pushing elements apart

Pro Tips üí°

  • üéØ Negative margins can overlap elements
  • üéØ margin: auto only works horizontally by default
  • üéØ Padding percentages are based on width (even vertical!)
  • üéØ outline doesn't affect layout (unlike border)
  • üéØ Use DevTools to experiment with values
Debug Tip: Can't figure out spacing? Add temporary background colors!

Avoid These Mistakes! ‚ö†Ô∏è

  • ‚ùå Forgetting box-sizing: border-box
  • ‚ùå Using padding when you need margin (or vice versa)
  • ‚ùå Not setting width when using margin: auto
  • ‚ùå Fighting margin collapse instead of working with it
  • ‚ùå Using only pixels - explore %, em, rem
  • ‚ùå Ignoring the DevTools box model viewer
