Part 1: Quick Review (Slides 4-6)

The Flex Shorthand

flex: 1 equals what three properties?
flex-grow: 1; flex-shrink: 1; flex-basis: 0;
Use hand gestures: grow (expand hands), shrink (compress), basis (flat hand for starting point)

Flex Properties Table

Students fill in purpose and default values for each property
‚Ä¢ flex-grow: Controls growing, default 0
‚Ä¢ flex-shrink: Controls shrinking, default 1
‚Ä¢ flex-basis: Starting size, default auto

Important Gotcha (Slide 6)

flex: 1 is NOT the same as flex-grow: 1
Why are they different?
flex: 1 sets basis to 0, flex-grow: 1 keeps basis at auto
Show this difference in VS Code - create two containers to compare

Part 2: Flexbox Axes (Slides 7-11)

Two Axes Concept (Slide 7)

Students define main axis and cross axis
‚Ä¢ Main Axis: primary direction of flow
‚Ä¢ Cross Axis: perpendicular to main axis
Use your arms to show axes - horizontal for main (row), vertical for cross

Flex Direction (Slide 8)

Default value of flex-direction?
row
Live code: Switch between row and column, show how items rearrange

Visual Example (Slide 9)

Have students draw axes in their notes for both row and column

Critical Concept (Slide 10)

When flex-direction changes, flex-basis reference changes!
In column direction, flex-basis refers to?
height (not width)

Common Pitfall (Slide 11)

flex: 1 in column direction can cause collapse with no height!
Why do items collapse?
flex: 1 means flex-basis: 0, and empty divs have 0 height
Show the collapse problem, then fix with flex: 1 1 auto or container height

COMMON STUDENT MISTAKES TO ADDRESS

‚Üí Nothing works without this! Always check first when debugging.

Confusing justify-content and align-items

‚Üí Use the "JUSTify = JUST the main axis" memory trick

Forgetting that column changes axis behavior

‚Üí Draw the axes every time when learning

Using flex: 1 without understanding flex-basis: 0

‚Üí Show the difference between flex: 1 and flex-grow: 1

Not setting height on column containers

‚Üí Column containers need explicit height or content

GUIDED NOTES ANSWER KEY SUMMARY

Part 1: Quick Review

  • flex: 1 = flex-grow: 1, flex-shrink: 1, flex-basis: 0
  • flex-grow default: 0
  • flex-shrink default: 1
  • flex-basis default: auto

Part 2: Axes

  • flex-direction default: row
  • Row: main = horizontal, cross = vertical
  • Column: main = vertical, cross = horizontal
  • In column, flex-basis refers to height

Part 3: Alignment

  • justify-content: main axis
  • align-items: cross axis
  • Perfect center: justify-content: center; align-items: center;
  • gap adds space between items only

Part 4: Patterns

  • Nav: space-between + center
  • Cards: flex: 1 1 300px
  • Equal columns: flex: 1
  • Prevent shrink: flex-shrink: 0
