Common Student Questions

Question Answer
Why can't we just use the VS Code shortcut? Building muscle memory; won't always have VS Code
What if I forget the DOCTYPE? Page may render in quirks mode; always validate
Can I use a different lang? Yes! lang="es" for Spanish, "fr" for French, etc.
Why index.html specifically? Web server default; automatically serves this file

If Running Behind Schedule

  • Demo validation quickly, let them explore in homework

  • Preview next class content briefly
