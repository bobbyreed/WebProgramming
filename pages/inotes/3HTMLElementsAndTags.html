TROUBLESHOOTING GUIDE

Common Student Issues & Solutions:

Issue Solution
HTML not displaying correctly Check: File saved? Browser refreshed? Correct file open?
Tags showing in browser Missing angle brackets or typo in tag name
VS Code not recognizing HTML File must end in .html, check file extension
Git push failing Need to pull first, or not logged in to GitHub
Can't find their repository Check GitHub profile, might be in different folder locally

KEY TALKING POINTS BY TOPIC

HTML vs CSS vs JavaScript:

  • "HTML is the noun, CSS is the adjective, JavaScript is the verb"
  • "You could build a website with just HTML - ugly but functional"
  • "CSS without HTML is like clothes without a body"

Elements and Tags:

  • "Tags are like quotation marks - they come in pairs (usually)"
  • "The browser reads HTML top to bottom, like a book"
  • "Think of elements as boxes or containers"

Semantic HTML:

  • "Google needs to understand your content to rank it"
  • "15% of web users rely on screen readers"
  • "Future you will thank current you for semantic HTML"

Void Elements:

  • "Some elements ARE the content, not containers"
  • "Think of br as pressing Enter on your keyboard"
  • "hr is like drawing a line across your page"
