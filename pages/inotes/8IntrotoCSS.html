Slide 7: CSS Basic Syntax

üîë GUIDED NOTES ANSWERS - Students Fill In:

  • Basic syntax: selector { property: value; }
  • Properties and values separated by : (colon)
  • Declarations end with ; (semicolon)
LIVE CODE EXAMPLE: p { color: red; /* point out each part */ font-size: 16px; /* no space before px! */ }
EMPHASIZE: Semicolons are CRITICAL! Missing semicolon = broken CSS. This WILL be on the test!

üìç SELECTORS (Slides 9-18)

Slides 9-13: Basic Selectors

üîë GUIDED NOTES ANSWERS - Selector Table:

Selector Type Symbol What it Selects
Universal * Everything on the page
Type (none) All elements of that type
Class . (period) Elements with that class
ID # (hashtag) The ONE element with that ID
QUICK ACTIVITY: Write on board:
  • .warning
  • #header
  • p
Ask: "Which symbol for which type?"
KEY POINT for Notes:
  • Classes can be used multiple times
  • IDs can only be used once per page
Slides 15-17: Advanced Selectors

üîë GUIDED NOTES ANSWERS:

  • Grouping: Applies same styles to multiple selectors
  • Chaining: Elements must have ALL/both specified criteria
  • Chaining syntax: .alert.warning (no space!)
  • Descendant: Selects elements nested inside/within other elements
  • Descendant syntax: .container (space) p
Teaching Tip: CRITICAL distinction -
  • NO SPACE = chaining (must have both)
  • SPACE = descendant (one inside the other)
Write both on board and circle the space/no-space

üìç CSS PROPERTIES (Slides 19-21)

Slides 19-20: Color & Typography

üîë GUIDED NOTES ANSWERS:

  • color property changes: text color
  • background-color changes: background color
  • font-family sets: font/typeface
  • font-size sets: size of text
  • font-weight sets: boldness
  • text-align aligns text: horizontally
LIVE DEMO: p { color: blue; /* text color */ background-color: yellow; font-family: Arial, sans-serif; /* ALWAYS have fallback! */ font-size: 16px; /* NO SPACE before px */ font-weight: bold; /* or use 700 */ }
COMMON MISTAKE: Students often write "font-size: 16 px;" with a space. This breaks! Emphasize: NO SPACE before units!

üìç ADDING CSS TO HTML (Slides 22-25)

Three Methods Overview

üîë GUIDED NOTES ANSWERS:

Method Where CSS Goes
External Separate .css file
Internal In <style> tags in <head>
Inline In style attribute on element

üîë External CSS Link Syntax:

  • <link rel="stylesheet" href="styles.css">

üîë Internal CSS:

  • Uses <style> tags

üîë Inline CSS:

  • Uses style="..." attribute
STRESS THIS: External CSS is BEST PRACTICE! Use for all projects. Internal = sometimes okay. Inline = avoid!

üìç KEY CONCEPTS REVIEW ANSWERS

Fill-in-the-Blank Review

üîë Complete Answer Key:

  1. CSS stands for Cascading Style Sheets
  2. Three parts of CSS rule: selector, property, value
  3. To select a class: . (period)
  4. To select an ID: # (hashtag)
  5. Best way to add CSS: External
  6. Classes: multiple times, IDs: one time
  7. Multiple selectors separated with: , (comma)
  8. Nested elements use: space/descendant combinator

‚ö†Ô∏è COMMON STUDENT ISSUES TO WATCH FOR

Troubleshooting Guide
  • Forgetting semicolons: CSS stops working - check this first!
  • Space before px: "16 px" doesn't work, must be "16px"
  • Wrong selector symbol: Using # for class or . for ID
  • Confusing chaining vs descendant: No space vs space
  • Not saving files: Remind to save CSS file before refreshing browser
  • Wrong file path: href="styles.css" assumes same folder
  • Case sensitivity: Class names are case-sensitive!
