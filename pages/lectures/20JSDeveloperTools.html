Opening DevTools 🚀

Three Ways to Access

Method 1: Chrome Menu

Menu → More Tools → Developer Tools

Method 2: Right-Click

Right-click anywhere → Select "Inspect"

Method 3: Keyboard Shortcuts

  • Windows/Linux: F12 or Ctrl + Shift + C
  • Mac: Opt + Cmd + C

Let's try it! Open DevTools using each method

Live DOM Manipulation 🎭

Using Console to Change the Page

// Select elements
document.querySelector('h1');
document.querySelectorAll('.button');

// Modify elements
document.querySelector('h1').textContent = "Changed!";
document.querySelector('h1').style.color = "red";

// Add new elements
const newDiv = document.createElement('div');
newDiv.textContent = "I'm new!";
document.body.appendChild(newDiv);

JavaScript Debugging 🐛

Sources Panel - Your Debug HQ

Debugging Workflow:

  1. Open Sources panel
  2. Find your JavaScript file
  3. Set breakpoints (click line numbers)
  4. Reload page to hit breakpoint
  5. Step through code line by line
  6. Inspect variables in Scope panel
No more console.log debugging! Use breakpoints instead

Breakpoints 🎯

Pause Your Code Where You Need It

Types of Breakpoints:

  • Line breakpoints
  • Conditional breakpoints
  • DOM breakpoints
  • XHR/Fetch breakpoints
  • Event listener breakpoints

Debug Controls:

  • ▶️ Resume execution
  • ⤵️ Step over
  • ⬇️ Step into
  • ⬆️ Step out
  • 🔄 Restart
function calculate(x, y) {
  const sum = x + y; // Set breakpoint here
  const product = x * y;
  return { sum, product };
}

Watch & Scope 👁️

Monitor Variables While Debugging

Scope Panel Shows:

  • Local: Current function variables
  • Closure: Variables from outer functions
  • Global: Window object properties

Watch Expressions:

  • Add any JavaScript expression
  • Updates as you step through code
  • Great for tracking specific values

Example: Watch x + y to see sum update

DevTools Performance Tips ⚡

Quick Wins:

  • 🔍 Use console.time() to measure execution time
  • 📊 Network tab shows slow requests
  • 🎨 Rendering tab shows paint flashing
  • 💾 Memory tab identifies leaks
  • ⚡ Lighthouse for overall audit
console.time('fetchData');
await fetch('/api/data');
console.timeEnd('fetchData'); // Shows: fetchData: 234.56ms
