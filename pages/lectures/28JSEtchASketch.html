Targeting Elements üéØ

Query Selectors

// Select by ID
const container = document.querySelector("#container");

// Select by class
const display = document.querySelector(".display");

// Select by tag
const divs = document.querySelectorAll("div");

// Complex selectors (CSS-style)
const button = document.querySelector("#container > button.primary");
Remember: querySelector returns ONE element, querySelectorAll returns a NodeList (like an array, but not quite!)

Using Relationships

Creating Elements üèóÔ∏è

Building DOM Nodes in Memory

// Step 1: Create element in memory
const newDiv = document.createElement("div");

// Step 2: Add content and attributes
newDiv.textContent = "Hello World!";
newDiv.classList.add("message");
newDiv.id = "welcome";

// Step 3: Add to the DOM
container.appendChild(newDiv);

Key Concept: Elements are created in memory first, then added to the page!

Class Manipulation üé®

The classList API

const box = document.querySelector(".box");

// Add a class
box.classList.add("highlight");

// Remove a class
box.classList.remove("old-style");

// Toggle a class (add if missing, remove if present)
box.classList.toggle("active");

// Check if class exists
if (box.classList.contains("special")) {
  console.log("It's special!");
}
Best Practice: Toggle classes instead of inline styles for cleaner code!

Responding to User Actions

Events are: Things that happen on your webpage (clicks, key presses, mouse movements, etc.)

Common Events:

click dblclick mouseover mouseout keydown keyup submit load scroll focus

Three Ways to Add Events 3Ô∏è‚É£

‚ùå Method 1: HTML Attribute

<button onclick="alert('Hi!')">Click</button>

Mixes JS with HTML, only one handler

‚ö†Ô∏è Method 2: DOM Property

Better, but still only one handler

‚úÖ Method 3: addEventListener

  alert("Hi!");
});

Clean, multiple handlers possible, most flexible!

addEventListener() üéØ

The Right Way‚Ñ¢

const button = document.querySelector("#myButton");

// Anonymous function
button.addEventListener("click", () => {
  console.log("Clicked!");
});

// Named function (reusable!)
function handleClick() {
  console.log("Button was clicked!");
}
button.addEventListener("click", handleClick);

// With event object
button.addEventListener("click", (e) => {
  console.log(e.target); // The clicked element
  e.target.style.background = "blue";
});

The Event Object üì¶

Information About What Happened

button.addEventListener("click", function(e) {
  // e is the event object
  console.log(e.type); // "click"
  console.log(e.target); // Element that was clicked
  console.log(e.currentTarget); // Element with listener
  console.log(e.clientX); // Mouse X position
  console.log(e.clientY); // Mouse Y position
  console.log(e.timeStamp); // When it happened
});

// Prevent default behavior
link.addEventListener("click", (e) => {
  e.preventDefault(); // Stop link navigation
});

Events on Multiple Elements üé≠

Using forEach with NodeLists

// Select ALL buttons
const buttons = document.querySelectorAll("button");

// Add listener to EACH button
buttons.forEach((button) => {
  button.addEventListener("click", () => {
    alert(`You clicked ${button.textContent}`);
    button.classList.toggle("clicked");
  });
});

// Or use the event target to identify which one
buttons.forEach((button) => {
  button.addEventListener("click", (e) => {
    e.target.style.background = "yellow";
  });
});

Event Bubbling ü´ß

Events Travel Up the DOM Tree

When you click a child element, the event "bubbles" up through all parent elements!

<div id="outer"> <-- 3rd: Event bubbles here
  <div id="middle"> <-- 2nd: Then here
    <button id="inner">Click</button> <-- 1st: Click happens
  </div>
</div>

// Stop bubbling with stopPropagation()
button.addEventListener("click", (e) => {
  e.stopPropagation();
  console.log("Button clicked, bubbling stopped!");
});

Creating the Grid üî≤

JavaScript Logic

function createGrid(size) {
  const container = document.querySelector('#container');
  
  // Clear existing grid
  container.innerHTML = '';
  
  // Create size x size grid
  for (let i = 0; i < size * size; i++) {
    const square = document.createElement('div');
    square.classList.add('grid-square');
    
    // Add hover effect
    square.addEventListener('mouseenter', () => {
      square.style.backgroundColor = 'black';
    });
    
    container.appendChild(square);
  }
}

// Start with 16x16 grid
createGrid(16);
