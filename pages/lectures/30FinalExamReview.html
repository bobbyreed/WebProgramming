CSS Specificity Rules

Specificity Values:

  • Inline styles: 1000 points
  • ID selectors: 100 points
  • Class selectors: 10 points
  • Element selectors: 1 point
/* Calculate specificity */
#header .nav li { } /* 100 + 10 + 1 = 111 */
.content p { } /* 10 + 1 = 11 */
p { } /* 1 */

Higher specificity always wins, regardless of order!

Display Property Values

Block Elements

  • Take full width available
  • Start on new line
  • Can contain other blocks
  • Examples: div, p, h1-h6

Inline Elements

  • Only take needed width
  • Stay in line with text
  • Can't set width/height
  • Examples: span, a, strong

Flexbox Layout System

/* Make a flex container */
.container {
  display: flex;
  flex-direction: row; /* or column */
  justify-content: center; /* main axis */
  align-items: center; /* cross axis */
}

/* Flex items */
.item {
  flex: 1; /* grow, shrink, basis */
}
Perfect Centering: justify-content: center; + align-items: center;

Variables: let vs const

const userName = "Alice";

score = score + 10; // This is allowed

Best Practice: Use const by default, let only when needed

JavaScript Functions

  return `Hello, ${name}!`;
}

// Function call

// Arrow function
const add = (a, b) => a + b;

// Functions can be stored in variables

  return x * y;
};

Parameters vs Arguments

Key Distinction:

  • Parameters: Variables in function definition (placeholders)
  • Arguments: Actual values passed when calling function
// 'a' and 'b' are PARAMETERS
function multiply(a, b) {
  return a * b;
}

// 5 and 3 are ARGUMENTS
multiply(5, 3); // Returns 15

Equality: == vs ===

== (Loose Equality)

Type coercion happens

"5" == 5 // true
0 == false // true
null == undefined // true

=== (Strict Equality)

Type and value must match

"5" === 5 // false
0 === false // false
null === undefined // false

Always use === for predictable comparisons!

Essential Array Methods

const numbers = [1, 2, 3, 4, 5];

// filter - creates new array with matching elements
const evens = numbers.filter(n => n % 2 === 0); // [2, 4]

// map - transforms each element
const doubled = numbers.map(n => n * 2); // [2, 4, 6, 8, 10]

// reduce - combines into single value
const sum = numbers.reduce((a, b) => a + b, 0); // 15

// forEach - loops through (no return)
numbers.forEach(n => console.log(n));

JavaScript Scope

// Block scope (let and const)
function myFunction() {
  let x = 10; // Function scope
  
  if (true) {
    let y = 20; // Block scope
    const z = 30; // Block scope
  }
  
  console.log(x); // 10 - works
  console.log(y); // Error! y not accessible
}

console.log(x); // Error! x not accessible outside function
