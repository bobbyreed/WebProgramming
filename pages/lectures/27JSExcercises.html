Exercise 1: repeatString 🔁

The Challenge:

Write a function that repeats a given string a specified number of times.

repeatString('hey', 3) // returns 'heyheyhey'
repeatString('hello', 10) // returns 'hellohellohello...'
repeatString('hi', 1) // returns 'hi'
repeatString('bye', 0) // returns ''
repeatString('goodbye', -1) // returns 'ERROR'
Hints:
  • Check for negative numbers first
  • Consider using String.repeat() or a loop
  • Don't forget edge cases!

Test Command: npm test repeatString

Exercise 2: reverseString 🔄

The Challenge:

Write a function that reverses a string.

reverseString('hello') // returns 'olleh'
reverseString('hello there') // returns 'ereht olleh'
reverseString('123! abc!') // returns '!cba !321'
reverseString('') // returns ''
Hints:
  • Strings are immutable - you'll need to create a new one
  • Consider split(), reverse(), and join()
  • Or try a loop working backwards!

Exercise 3: removeFromArray 🗑️

The Challenge:

Remove all given values from an array.

removeFromArray([1, 2, 3, 4], 3) // returns [1, 2, 4]
removeFromArray([1, 2, 3, 4], 3, 2) // returns [1, 4]
removeFromArray([1, 2, 3, 4], 7, "tacos") // returns [1, 2, 3, 4]
removeFromArray([1, 2, 3, 4], 7, 2) // returns [1, 3, 4]
removeFromArray(["hey", 2, 3, "ho"], "hey", 3) // returns [2, "ho"]
Hints:
  • The function can receive any number of values to remove
  • Look into the arguments object or rest parameters ...
  • filter() might be your friend here

Exercise 4: sumAll ➕

The Challenge:

Sum all numbers in a range between two numbers (inclusive).

sumAll(1, 4) // returns 10 (1 + 2 + 3 + 4)
sumAll(1, 4000) // returns 8002000
sumAll(123, 1) // returns 7626 (works backwards too!)
sumAll(-10, 4) // returns 'ERROR' (no negative numbers)
sumAll(10, "90") // returns 'ERROR' (numbers only)
sumAll(10, [90, 1]) // returns 'ERROR' (invalid input)
Hints:
  • Validate inputs first!
  • Handle the case where first > second
  • A simple loop works great here

Exercise 5: leapYears 📅

The Challenge:

Determine whether a given year is a leap year.

leapYears(2000) // returns true
leapYears(1985) // returns false
leapYears(1996) // returns true
leapYears(1900) // returns false
leapYears(1600) // returns true
leapYears(700) // returns false
Leap Year Rules:
  • Divisible by 4 → leap year
  • EXCEPT: Divisible by 100 → NOT a leap year
  • EXCEPT EXCEPT: Divisible by 400 → IS a leap year

Exercise 6: tempConversion 🌡️

The Challenge:

Two functions: Convert Fahrenheit ↔ Celsius (round to 1 decimal place).

convertToCelsius(32) // returns 0
convertToCelsius(100) // returns 37.8
convertToCelsius(-100) // returns -73.3

convertToFahrenheit(0) // returns 32
convertToFahrenheit(73.2) // returns 163.8
convertToFahrenheit(-10) // returns 14
Formulas:
  • C = (F - 32) × 5/9
  • F = C × 9/5 + 32
  • Round to 1 decimal: Math.round(num * 10) / 10
