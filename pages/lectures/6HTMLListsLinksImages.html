Lists Side by Side

Unordered List <ul>

Use when:

  • Order doesn't matter
  • Items are equally important
  • Shopping lists
  • Feature lists
  • Navigation menus

Ordered List <ol>

Use when:

  1. Sequence is important
  2. Step-by-step instructions
  3. Rankings or priorities
  4. Numbered references
  5. Chronological events
Remember: Both use <li> for individual items!

Absolute vs. Relative Links

Absolute Links ğŸŒ

Full web address including:

  • Protocol (https://)
  • Domain (www.site.com)
  • Path (/page.html)

Use for: External websites

Relative Links ğŸ“

Path relative to current file:

  • Same folder: page.html
  • Subfolder: folder/page.html
  • Parent: ../page.html

Use for: Your own site

Opening in New Tabs ğŸ”²

The target Attribute

<!-- Opens in same tab (default) -->
<a href="https://google.com">Google</a>

<!-- Opens in new tab -->
<a href="https://google.com" target="_blank">Google</a>

âš ï¸ Security Warning!

Always add rel="noopener noreferrer" when using target="_blank"

<a href="https://google.com" target="_blank" rel="noopener noreferrer">Google</a>

This prevents "tabnabbing" attacks!

Link Security ğŸ”’

Understanding rel Attribute Values

noopener

  • Prevents new page from accessing original page
  • Blocks window.opener
  • Security protection

noreferrer

  • Hides referrer information
  • Includes noopener behavior
  • Privacy + Security
Best Practice: Always use rel="noopener noreferrer" with target="_blank"
