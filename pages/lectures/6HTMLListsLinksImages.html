Lists in HTML 📝

Lists are EVERYWHERE on the web!

  • Navigation menus
  • Shopping lists
  • Recipe ingredients
  • Step-by-step instructions
  • Social media feeds
  • Comment sections
Two Types: Unordered (bullets) and Ordered (numbers)

Unordered Lists •

When order doesn't matter

<ul>
  <li>Milk</li>
  <li>Eggs</li>
  <li>Bread</li>
  <li>Coffee</li>
</ul>

This renders as:

  • Milk
  • Eggs
  • Bread
  • Coffee

Ordered Lists 1️⃣

When sequence matters!

<ol>
  <li>Preheat oven to 350°F</li>
  <li>Mix dry ingredients</li>
  <li>Add wet ingredients</li>
  <li>Bake for 25 minutes</li>
</ol>

This renders as:

  1. Preheat oven to 350°F
  2. Mix dry ingredients
  3. Add wet ingredients
  4. Bake for 25 minutes

Lists Side by Side

Unordered List <ul>

Use when:

  • Order doesn't matter
  • Items are equally important
  • Shopping lists
  • Feature lists
  • Navigation menus

Ordered List <ol>

Use when:

  1. Sequence is important
  2. Step-by-step instructions
  3. Rankings or priorities
  4. Numbered references
  5. Chronological events
Remember: Both use <li> for individual items!

Links: The Web in "World Wide Web" 🔗

Links connect everything!

Links are what make the web a "web" - they connect pages together!

Your Page

The Anchor Element: <a> creates all links!

The Anchor Element ⚓

Creating Links with <a>

<!-- Link without href (doesn't work) -->
<a>Click me</a>

<!-- Link with href (works!) -->
<a href="https://www.google.com">Go to Google</a>
Key Point: The href attribute tells the link WHERE to go!
Without href, it's just text!

HTML Attributes 🏷️

Adding Information to Elements

Attributes provide additional information about HTML elements

<tagname attribute="value">content</tagname>

Common Attributes We'll Use:

  • href - Where a link goes
  • src - Source of an image
  • alt - Alternative text for images
  • target - Where to open the link
  • rel - Relationship to linked document

Absolute vs. Relative Links

Absolute Links 🌐

Full web address including:

  • Protocol (https://)
  • Domain (www.site.com)
  • Path (/page.html)
<a href="https://www.google.com">Google</a>

Use for: External websites

Relative Links 📁

Path relative to current file:

  • Same folder: page.html
  • Subfolder: folder/page.html
  • Parent: ../page.html
<a href="about.html">About</a>

Use for: Your own site

Navigating File Paths 🗂️

project/
├── index.html
├── about.html
├── pages/
│ ├── contact.html
│ └── portfolio.html
└── images/
    └── logo.png

From index.html:

  • To about.html: href="about.html" or href="./about.html"
  • To contact.html: href="pages/contact.html"
  • To logo.png: src="images/logo.png"

From pages/contact.html:

  • To index.html: href="../index.html"
  • To portfolio.html: href="portfolio.html"
  • To logo.png: src="../images/logo.png"

Opening in New Tabs 🔲

The target Attribute

<!-- Opens in same tab (default) -->
<a href="https://google.com">Google</a>

<!-- Opens in new tab -->
<a href="https://google.com" target="_blank">Google</a>

⚠️ Security Warning!

Always add rel="noopener noreferrer" when using target="_blank"

<a href="https://google.com" target="_blank" rel="noopener noreferrer">Google</a>

This prevents "tabnabbing" attacks!

Link Security 🔒

Understanding rel Attribute Values

noopener

  • Prevents new page from accessing original page
  • Blocks window.opener
  • Security protection

noreferrer

  • Hides referrer information
  • Includes noopener behavior
  • Privacy + Security
Best Practice: Always use rel="noopener noreferrer" with target="_blank"

Images in HTML 📷

Adding Visual Content

<img src="path/to/image.jpg" alt="Description of image">
Important: <img> is a void element - no closing tag!

Required Attributes:

  • src - Path to the image file
  • alt - Alternative text description

Image Attributes 🖼️

<img src="images/cat.jpg" alt="Orange tabby cat sleeping" width="400" height="300">

Why alt text matters:

  • 🦮 Screen readers for visually impaired users
  • 🚫 Displays when image fails to load
  • 🔍 Helps search engines understand content
  • 📱 Shows on slow connections
Pro Tip: Always include descriptive alt text - it's required for accessibility!

Image Formats 🎨

Choosing the Right Format

Format Best For Features
JPEG/JPG Photos Compressed, smaller files, no transparency
PNG Graphics, logos Transparency support, larger files
GIF Simple animations Animation, limited colors
SVG Icons, logos Vector, scalable, tiny files
WebP Modern web Best compression, not universal support

Organizing Images 📁

Best Practices for File Structure

project/
├── index.html
├── pages/
│ ├── about.html
│ └── contact.html
└── images/
    ├── logo.png
    ├── hero-banner.jpg
    ├── team/
    │ ├── john.jpg
    │ └── jane.jpg
    └── products/
        ├── product1.jpg
        └── product2.jpg
Tips:
  • Keep all images in an images folder
  • Use subfolders for organization
  • Use descriptive file names (not IMG_1234.jpg)

  </ol>

</article>
