Quick Review: Array Methods 🔄

Building Blocks We'll Use Today

Remember: These methods don't change the original array - they return new arrays!

The Big Four:

filter(): Keep items that pass a test
map(): Transform every item
sort(): Arrange items in order
reduce(): Boil down to a single value

Exercise 1: Filter 🔍

Filter inventors born in the 1500s

Your Task:

Use filter() to find all inventors born between 1500-1599

// Starter code
const fifteen = inventors.filter(inventor => {
  // What condition goes here?
});

// Hint: Check if year is >= 1500 AND < 1600

Solution: Filter ✅

const fifteen = inventors.filter(inventor =>
  inventor.year >= 1500 && inventor.year < 1600
);

console.table(fifteen);

// Results: Galileo, Kepler, and Copernicus!
Pro Tip: console.table() displays objects in a nice table format!

Exercise 2: Map 🗺️

Create array of inventor full names

Your Task:

Transform the inventors array into an array of full names

// Starter code
const fullNames = inventors.map(inventor => {
  // Combine first and last names
});

// Expected: ["Albert Einstein", "Isaac Newton", ...]

Solution: Map ✅

const fullNames = inventors.map(inventor =>
  `${inventor.first} ${inventor.last}`
);

console.log(fullNames);
// ["Albert Einstein", "Isaac Newton", "Galileo Galilei", ...]
Template Literals Rock! The backticks (`) let us embed variables easily.

Exercise 3: Sort 📊

Sort inventors by birthdate (oldest to youngest)

Your Task:

Arrange inventors chronologically by birth year

// Remember: sort() compares two items at a time
const ordered = inventors.sort((a, b) => {
  // Return 1 if a should come after b
  // Return -1 if a should come before b
});

Solution: Sort ✅

// Long form
const ordered = inventors.sort((a, b) => {
  if(a.year > b.year) return 1;
  else return -1;
});

// Short form with ternary
const ordered = inventors.sort((a, b) =>
  a.year > b.year ? 1 : -1
);

console.table(ordered);

Exercise 4: Reduce 🔢

Total years lived by all inventors

Your Task:

Calculate the sum of all inventors' lifespans

// reduce() accumulates values
const totalYears = inventors.reduce((total, inventor) => {
  // Add this inventor's lifespan to total
}, 0); // Start with 0

Solution: Reduce ✅

const totalYears = inventors.reduce((total, inventor) => {
  return total + (inventor.passed - inventor.year);
}, 0);

console.log(totalYears); // 861 years!
Pattern: reduce((accumulator, current) => newValue, initialValue)

Exercise 5: Sort by Years Lived 🏆

Who lived the longest?

const oldest = inventors.sort((a, b) => {
  const lastGuy = a.passed - a.year;
  const nextGuy = b.passed - b.year;
  return lastGuy > nextGuy ? -1 : 1;
});

console.table(oldest);

Notice: We calculate lifespan on the fly during sorting!

Exercise 7: Sort People Alphabetically 🔤

Handle "Last, First" format

const people = ['Bernhard, Sandra', 'Bethea, Erin', ...];

const alpha = people.sort((lastOne, nextOne) => {
  const [aLast, aFirst] = lastOne.split(', ');
  const [bLast, bFirst] = nextOne.split(', ');
  return aLast > bLast ? 1 : -1;
});

console.log(alpha);
Destructuring: [aLast, aFirst] pulls out array values!

Exercise 8: Count Instances 📊

Build an object with counts

const data = ['car', 'car', 'truck', 'truck', 'bike', ...];

const transportation = data.reduce((obj, item) => {
  if (!obj[item]) {
    obj[item] = 0;
  }
  obj[item]++;
  return obj;
}, {});

console.log(transportation);
// {car: 5, truck: 3, bike: 2, ...}

Discovery Methods

New Data Set:

const people = [
  { name: 'Wes', year: 1988 },

  { name: 'Irv', year: 1970 },
  { name: 'Lux', year: 2015 }
];

const comments = [
  { text: 'Love this!', id: 523423 },
  { text: 'Super good', id: 823423 },
  // ... more comments
];

some() Method 🔍

Is at least one person 19 or older?

// some() returns true if ANY element passes the test
const isAdult = people.some(person => {
  const currentYear = (new Date()).getFullYear();
  return currentYear - person.year >= 19;
});

// Shorter version
const isAdult = people.some(person =>
  ((new Date()).getFullYear()) - person.year >= 19
);

console.log({isAdult}); // true

every() Method ✅

Is everyone 19 or older?

// every() returns true only if ALL elements pass
const allAdults = people.every(person =>
  ((new Date()).getFullYear()) - person.year >= 19
);

console.log({allAdults}); // false (Lux is too young!)

some()

At least ONE matches

every()

ALL must match

find() Method 🎯

Find comment with ID 823423

// find() returns the FIRST matching element
const comment = comments.find(comment =>
  comment.id === 823423
);

console.log(comment);
// { text: 'Super good', id: 823423 }
find() vs filter():
• filter() returns an array of ALL matches
• find() returns ONLY the first match

findIndex() Method 📍

Find and delete a comment

// findIndex() returns the position of the first match
const index = comments.findIndex(comment =>
  comment.id === 823423
);

console.log(index); // 1

// Two ways to delete:
// 1. Mutate original array
comments.splice(index, 1);

// 2. Create new array (immutable)
const newComments = [
  ...comments.slice(0, index),
  ...comments.slice(index + 1)
];

The Spread Operator ✨

Building new arrays without mutation

// The ... spread operator "unpacks" arrays
const newComments = [
  ...comments.slice(0, index), // Everything before
  ...comments.slice(index + 1) // Everything after
];

// This creates a NEW array without changing the original!

Why Avoid Mutation?

• Easier to debug
• Safer in complex apps
• Required for React/Redux
• Better for undo/redo features

Array Methods Cheat Sheet 📋

Method Returns Use When
filter() New array Keep matching items
map() New array Transform all items
sort() Same array Order items
reduce() Single value Calculate from all
some() Boolean Test if any match
every() Boolean Test if all match
find() Single item Get first match
findIndex() Number Get position of match

Debugging Array Methods 🐛

Pro Debugging Tips:

// 1. Use console.table() for objects
console.table(inventors);

// 2. Check intermediate steps
const filtered = data.filter(item => {
  console.log('Checking:', item); // Debug line
  return item.value > 10;
});

// 3. Break complex chains
const step1 = data.filter(...);
console.log('After filter:', step1);
const step2 = step1.map(...);
console.log('After map:', step2);

// 4. Use descriptive variable names
const adultUsers = users.filter(user => user.age >= 18);

Performance Tips ⚡

Write efficient array code

❌ Less Efficient

// Multiple passes
const adults = users.filter(u => u.age >= 18);
const names = adults.map(u => u.name);
const sorted = names.sort();

✅ More Efficient

// Chain when possible
const result = users
  .filter(u => u.age >= 18)
  .map(u => u.name)
  .sort();
But remember: Readability often matters more than micro-optimizations!
