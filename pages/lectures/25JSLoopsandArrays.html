For Loops ğŸ¢

When you know how many times to repeat

// for (initialization; condition; update)
for (let i = 0; i < 5; i++) {
  console.log(`Iteration ${i}`);
}

Parts of a For Loop:

  1. Initialization: let i = 0
  2. Condition: i < 5
  3. Update: i++

Execution Order:

  1. Initialize once
  2. Check condition
  3. Execute body
  4. Update
  5. Go to step 2

Loop Control: Break & Continue ğŸ›‘

break

Exits the loop completely

for (let i = 0; i < 10; i++) {
  if (i === 5) break;
  console.log(i);
}
// Output: 0, 1, 2, 3, 4

continue

Skips to next iteration

for (let i = 0; i < 5; i++) {
  if (i === 2) continue;
  console.log(i);
}
// Output: 0, 1, 3, 4

More Elegant Than Loops!

The Big Three:

Key Concept: These methods take functions as arguments (callbacks)

The Map Method ğŸ—ºï¸

Transform Every Element

// Double all numbers
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map(num => num * 2);
console.log(doubled); // [2, 4, 6, 8, 10]

Important: map() returns a NEW array - original is unchanged!

The Filter Method ğŸ”

Keep Only What Passes the Test

// Keep only even numbers
const numbers = [1, 2, 3, 4, 5, 6, 7, 8];
const evens = numbers.filter(num => num % 2 === 0);
console.log(evens); // [2, 4, 6, 8]

// Keep long words
const words = ["hi", "hello", "goodbye", "yes", "no"];
const longWords = words.filter(word => word.length > 3);
console.log(longWords); // ["hello", "goodbye"]

Filter returns elements where callback returns true

The Reduce Method ğŸ“Š

Combine All Elements Into One Value

// Sum all numbers
const numbers = [1, 2, 3, 4, 5];
const sum = numbers.reduce((total, num) => total + num, 0);
console.log(sum); // 15

// Find the maximum
const scores = [85, 92, 78, 96, 88];
const max = scores.reduce((max, score) => {
  return score > max ? score : max;
}, 0);
console.log(max); // 96
Pattern: reduce((accumulator, current) => ..., initialValue)

Let's Code: sumOfTripledEvens ğŸ’ª

10-Minute Challenge

Write a function that:

  1. Takes an array of numbers
  2. Filters for even numbers
  3. Triples each even number
  4. Sums the results
// Example:
sumOfTripledEvens([1, 2, 3, 4, 5, 6]);
// Should return: 36 (2*3 + 4*3 + 6*3)

Solution: Three Approaches ğŸ¯

Traditional Loop

function sumOfTripledEvens(arr) {
  let sum = 0;
  for (let i = 0; i < arr.length; i++) {
    if (arr[i] % 2 === 0) {
      sum += arr[i] * 3;
    }
  }
  return sum;
}

Array Methods âœ¨

function sumOfTripledEvens(arr) {
  return arr
    .filter(n => n % 2 === 0)
    .map(n => n * 3)
    .reduce((sum, n) => sum + n, 0);
}

Common Array Patterns ğŸ“‹

const arr = [1, 2, 3, 4, 5];

// Find if ANY element matches

// Check if ALL elements match

// Find first matching element

// Find index of element

// Check if includes
arr.includes(3); // true

Introduction to TDD ğŸ§ª

Write Tests First, Code Second

The TDD Cycle:

  1. ğŸ”´ Red: Write a failing test
  2. ğŸŸ¢ Green: Write code to pass the test
  3. ğŸ”„ Refactor: Improve the code

Why TDD?

  • Ensures your code works correctly
  • Makes debugging easier
  • Documents expected behavior
  • Prevents regression bugs

TDD Example âœ…

// 1. Write the test first (it will fail!)
function testAdd() {
  const result = add(2, 3);
  if (result !== 5) {
    console.error("Test failed!");
  } else {
    console.log("Test passed!");
  }
}

// 2. Write code to make it pass
function add(a, b) {
  return a + b;
}

// 3. Run the test
testAdd(); // "Test passed!"
